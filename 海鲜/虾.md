# 虾

## 白灼虾

1. 食材
   1. 葱
   2. 姜
   3. 料酒/白酒
   4. 虾
   5. 冰水

2. 做法

    ```text
    1. 加清水，放入葱、姜、酒，大火煮沸
    2. 水沸后倒入虾，保持大火
    3. 虾身变红、弯曲成 C 形即可
    4. 捞出虾后立刻放入冰水浸泡 3 分钟

    ----------

    1. 加清水，放入葱、姜，大火煮沸
    2. 水沸后倒入虾，保持大火，煮 10 秒后关火
    3. 再 30 秒后加入适量糖，提鲜
    4. 再 30 秒后加入适量盐
    5. 再 30 秒后捞出
    ```
