# kitchen

## 爆香

1. 热锅冷油
   1. 小火，干香料
   2. 中火，花椒/干辣椒，至辣椒微棕即可
   3. 大火，葱/姜/蒜，蒜末变得金黄、姜片边缘微卷、葱段开始焦黄结束

## 盐和糖

1. 腌制时，比例控制
   1. 脱水为主，2.5 克盐 + 1 克糖
   2. 调味为主，1 克盐 + 1 克糖

## 焯水

1. 肉类
   1. 冷水
   2. 料酒、姜片、葱段去腥
   3. 温水清洗干净
2. 蔬菜
   1. 沸水
   2. 少许盐和油（沸水后，再加盐）
   3. 捞出后立即过凉，保持爽脆口感和鲜艳色泽

## 生抽和老抽

1. 生抽原料：大豆、小麦、食盐、水，生抽是酱油的一种，专门用来代替味精、提鲜调味；入口咸味明显，鲜味很足
2. 老抽：在生抽的基础上，加入焦糖色，质地比生抽粘稠，摇晃后容易挂杯
